A simple and interactive Tic-Tac-Toe game built to demonstrate core programming concepts, game logic, and user interaction. Players take turns placing their marks (X and O) on a 3×3 grid, with the game automatically detecting wins, losses, and draws.

Features
Two-player gameplay
Win and draw detection
Interactive and user-friendly interface
Game restart/reset functionality
Clean and modular code structure
Technologies Used
C++.
How to Play
Start the game.
Players alternate turns placing X and O on the grid.
The first player to align three marks horizontally, vertically, or diagonally wins.
If all cells are filled without a winner, the game ends in a draw.
Reset the board to play again.
Purpose

This project was created as a beginner-friendly exercise to practice programming fundamentals such as conditional logic, state management, event handling, and user interface design.
